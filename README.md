🚀 SaaS Analytics Platform

A production-ready multi-tenant SaaS application built with Next.js 16.

✨ Features
Multi-tenant workspace system
Secure authentication (NextAuth)
Stripe subscription billing
Analytics dashboard (events, revenue, engagement)
Role-based team management
Notifications & activity logs
🛠 Tech Stack
Next.js 16 + React 19
TypeScript
PostgreSQL + Prisma
Tailwind CSS
Stripe API
⚙️ Setup
git clone https://github.com/yourusername/saas-analytics-platform.git
cd saas-analytics-platform
npm install

Setup .env using .env.example

npx prisma generate
npx prisma migrate dev
npm run dev
